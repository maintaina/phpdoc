Tags

Return values
array<string|int, mixed> —

An array of folder information. Each entry is keyed by the mailbox UTF-8 name and contains:

  • level: How many parents a folder has, 0 is the root.
  • label: The display label for the mailbox.
  • d: The delimiter.
