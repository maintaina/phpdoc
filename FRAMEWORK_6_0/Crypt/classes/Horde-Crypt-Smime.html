Parameters

  • 'certs': Additional signing certs (Optional)
  • 'passphrase': Passphrase for key (REQUIRED)
  • 'privkey': Private key (REQUIRED)
  • 'pubkey': Public key (REQUIRED)
  • 'sigtype': Determine the signature type to use. (Optional):
