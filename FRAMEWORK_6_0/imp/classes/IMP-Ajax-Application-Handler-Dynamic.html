autocompleteSearch()

Dynamic.php : 1331

AJAX Action: Do an autocomplete search.

public autocompleteSearch() : object

Variables used:

  • limit: (integer) If set, limits to this many results.
  • search: (string) Search string.
  • type: (string) Autocomplete search type.
Return values
object —

An object with a single property: 'results'. The format of 'results' depends on the search type.

  • type = 'email' Results is an array with the following keys for each result:
    • g: (array) List of addresses in the group (in same 'results' format as type = 'email').
    • l: (string) Full label.
    • s: (string) Short display string.
    • v: (string) Value.
